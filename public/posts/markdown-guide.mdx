# Markdown Guide

Welcome to the Markdown Guide! This post will help you understand the basics of Markdown syntax.

## Headers

You can create headers using the `#` symbol:

```markdown
# H1

## H2

### H3

#### H4

##### H5

###### H6
```

---

## Lists

### Unordered Lists

```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

### Ordered Lists

```markdown
1. First item
2. Second item
3. Third item
```

### Mixed Lists

```markdown
1. Ordered item
   - Unordered subitem 1
   - Unordered subitem 2
2. Another ordered item
```

---

## Links and Images

### Basic Link

```markdown
[Visit OpenAI](https://www.openai.com)
```

### Basic Image

```markdown
![Alt text for the image](/placeholder.svg?height=200&width=300)
```

### Link with Title

```markdown
[Visit OpenAI](https://www.openai.com "OpenAI's Homepage")
```

### Image with Title

```markdown
![Alt text with title](/placeholder.svg 'Image Title')
```

## Text Formatting

```markdown
**Bold text**

_Italic text_

~~Strikethrough~~

**_Combined bold and italic_**
```

## Blockquotes

### Basic Blockquote

```markdown
> This is a blockquote.  
> You can add multiple lines by breaking lines with two spaces.
```

### Nested Blockquotes

```markdown
> Outer quote
>
> > Nested quote
```

## Tables

### Basic Table

```markdown
| Syntax | Description | Example     |
| ------ | ----------- | ----------- |
| Header | Row Content | Row Example |
| `Code` | Inline      | `Example`   |
```

### Alignment Table

```markdown
| Left Align | Center Align | Right Align |
| ---------- | :----------: | ----------: |
| Item 1     |    Item 2    |      Item 3 |
```
